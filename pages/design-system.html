Guidelines: Primary blues drive brand identity and CTAs. Turquoise accent is used for highlights, gradients, focus rings, data viz, and subtle interactive affordances. Neutral palette is muted to let primary/accent stand out while improving light/dark theme parity. Use --success-muted, --warning-muted, --danger-muted, and --info-muted for soft background states instead of low opacity borders.

Accessibility: Primary 600 / Accent 600 on light backgrounds and Primary 300 / Accent 300 on dark backgrounds exceed 4.5:1 contrast vs body text surfaces. Avoid using Accent 400 or lighter for standalone text; reserve them for backgrounds, indicators, or gradient midpoints. Focus ring uses a semi-transparent accent glow (--focus-ring) to remain visible in both themes.

Token Policy: All production styles must reference design tokens. Raw hex values are confined to token declarations only. For ad-hoc usage leverage utility classes in css/utilities/theme-colors.css (e.g. .text-accent, .bg-accent-soft, .text-gradient).
